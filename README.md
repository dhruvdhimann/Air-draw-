## 🎯 About the Project

**Air Draw** is an interactive web-based drawing application that allows users to draw freely on a digital canvas, simulating the experience of drawing in the air. The project is built using **HTML, CSS, and JavaScript**, focusing on real-time interaction and smooth graphical rendering.

---

## ⚡ Features

- 🎨 **Freehand Drawing** – Smooth and responsive drawing  
- 🌈 **Color Picker** – Choose different colors  
- 🖌️ **Brush Size Control** – Adjustable stroke thickness  
- 🧹 **Clear Canvas** – Reset the drawing area instantly  
- 📱 **Responsive Design** – Works on desktop and mobile  

---

## 🧠 Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript** – Functionality  
- **Canvas API** – Drawing and rendering
